Flexbox レイアウト演習 — 確認用

Lesson 27 の5問を1ページにまとめました。見た目と、その下に「使ったCSS」が並んでいます。 見やすさのためボックスは 80px、コンテナの高さは 200px に縮めていますが、考え方は元の課題と同じです。

1横一列に並べる

何もしないと縦に積まれる div を、横方向に並べる。

.container {
    display: flex;
}

2中央に横並び

justify-content で横方向（主軸）の位置を中央に。

.container {
    display: flex;
    justify-content: center;
}

3均等な間隔で横並び

space-between は両端を端に付け、間を均等に空ける。

.container {
    display: flex;
    justify-content: space-between;
}
完全に均等にしたいときは space-evenly、各要素の左右に均等な余白なら space-around に変えてみてください。

4縦中央に横並び

align-items で縦方向（交差軸）の位置を中央に。上下に余白ができます。

.container {
    display: flex;
    align-items: center;
}

52×2 のグリッド

flex-wrap: wrap で折り返し、コンテナ幅を「2個分」にして2行に。

.container {
    display: flex;
    flex-wrap: wrap;
    width: 160px;  /* 80px × 2 でいっぱい */
}
幅が広いと折り返しが起きません。「ボックス2個でいっぱいになる幅」にするのがポイント。元の課題（100px のボックス）なら width: 200px です。
